## nix学习教程

NixOS 是一个基于 Nix 包管理器 的 Linux 发行版，以其独特的 声明式配置 和 纯函数式包管理 方式而闻名。与传统 Linux 发行版相比，NixOS 在系统管理、可复现性和回滚功能方面有着明显的差异。

nix包搜索网站：https://search.nixos.org/packages

nix中文社区官网：https://nixos-cn.org/
